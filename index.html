Twitter DID

The Twitter DID method is an implementation of the DID Core Specification that relies upon Twitter as a means of partial trust and resolution.

Twitter DIDs are distinguishable from other DID methods as they tie into an existing social media account, and often contain human-readable identifiers.

CLI

The DID Twitter Organization has created a CLI to facilitate simple generation, update, and deactivation of did:twitter documents. The CLI may further be expanded to hook into the Twitter API for DID management and resolution.
